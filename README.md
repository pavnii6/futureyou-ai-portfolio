## рџ’Ў How It Works

1. User enters a message.
2. The message is sent to the backend.
3. The backend creates a personalized prompt.
4. The AI model generates a response as the user's future self.
5. The response is displayed in the chat interface.
6. Conversation history is stored for future interactions.
