Project: NETRA Protect – AI-Based Glaucoma Risk Detection

Description:
Developed an AI-powered healthcare solution for early glaucoma detection using tear fluid biomarkers and diagnostic strip image analysis.

Key Features:
- HSV extraction from diagnostic strips
- Combined clinical + image-based features
- Multi-class classification (Normal, POAG, Suspect, Hypertension)
- Tkinter-based interactive UI
- Graph visualization + history tracking

Tech Stack:
- Python, XGBoost, Scikit-learn, OpenCV
- SMOTE for class balancing

Impact:
- Enables low-cost, early-stage glaucoma screening
- Reduces dependency on expensive diagnostic tools
